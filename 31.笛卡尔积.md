# 笛卡尔积

```javascript
function cartesianProduct (arr) {
  return arr.length === 1 ? arr : arr.reduce((a, c) => a.map(x => c.map(y => x.concat(y))).reduce((a, c) => a.concat(c), []), [[]])
}
// cartesianProduct([[1,2],[3,4,5]])
// [
//   [1,3],
//   [1,4],
//   [1,5],
//   [2,3],
//   [2,4],
//   [2,5]
// ]
```
