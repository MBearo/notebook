# stack

1. 私有属性
    1. 下划线
    2. symbol，但是可以用Object.getOwnPropertySymbols获取
    3. 通过WeakMap或者Map,但是无法继承私有属性。WeakMap可以回收。

    ```javascript
    const items = new WeakMap()
    class Stack {
      constructor () {
        items.set(this, [])
      }
      push (e) {
        const s = items.get(this)
        s.push(e)
      }
    }
    var a = new Stack()
    ```
